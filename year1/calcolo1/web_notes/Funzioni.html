Funzioni

la funzione è una relazione tra due insiemi AAA e BBB se ad ogni elemento di A si associa un solo elemento di BBB

L’inieme A è il Dominio della funzione e l’insieme B è il codominio della funzione.

Il Dominio quindi rappresenta quindi tutti i valori assumibili da xxx.

La funzione si scrive:

f:A⟶Bf: A \longrightarrow Bf:A⟶B

x⟼y=f(x)x \longmapsto y = f(x)x⟼y=f(x)

la xxx viene detta variabile indipendente mentre la yyy viene detta variabile dipendente.

Immagine è l’insieme dei valori assunti da y sul dominio DDD della funzione (si indica con f(D)f(D)f(D) ) Quindi le y che hanno una relazione con una x.

controimmagine è l’insieme dei valori assunti da x sul codominio BBB della funzione (si indica con f←(B)f^{\leftarrow}(B)f←(B) ) Quindi le x che hanno una relazione con una y.

L’insieme delle funzioni scritte nel modo y=mx+qy = mx+qy=mx+q rappresentano tutte le rette rappresentabili nel piano non verticali.

Es.

f:R⟶Rf: \mathbb{R} \longrightarrow \mathbb{R}f:R⟶R

f:x⟼y=2x+1f: x \longmapsto y = 2x+1f:x⟼y=2x+1

Dominio D=RD = \mathbb{R}D=R
Codominio B=RB = \mathbb{R}B=R
Immagine f(D)=Rf(D) = \mathbb{R}f(D)=R
controimmagine f←(B)=Rf^{\leftarrow}(B) = \mathbb{R}f←(B)=R

Funzioni iniettive e suriettive

Avendo
f:D⊆R⟶B⊆Rf: D \subseteq \mathbb{R} \longrightarrow B \subseteq \mathbb{R}f:D⊆R⟶B⊆R
x⟼y=f(x)x \longmapsto y = f(x)x⟼y=f(x)

  • fff è iniettiva se per ogni y∈By \in By∈B esiste al massimo un elemento x∈Dx \in Dx∈D
  • fff è suriettiva se per ogni y∈By \in By∈B esiste almeno un elemento x∈Dx \in Dx∈D
  • fff è biunivoca (o biiettiva) se per ogni y∈By \in By∈B esiste un solo un elemento x∈Dx \in Dx∈D. Quindi quando è sia iniettiva sia suriettiva.

Es.
y=xy = xy=x biunivoca (ad ogni x corrisponde una sola y e vice versa)
y=x2y = x^2y=x2 né iniettiva né suriettiva (ad ogni y possiamo trovare o una sola xxx (il vertice) o nessuna xxx o due xxx )

Interpretazione grafica

Tracciando ogni retta orizzontale appartenente al codominio allora:

  • Se la funzione viene sempre intersecata al massimo una volta è iniettiva.
  • Se la funzione viene sempre intersecata almeno una volta allora è suriettiva.
  • Se la funzione viene sempre intersecata una sola volta allora è biunivoca.

  4. biettiva se e solo se la funzione è sia iniettiva che suriettiva.

Funzioni monotone

Le funzioni monotono si dicono tali se sono:

  • Funzioni crescenti se scegliendo x1x_1x1​ e x2x_2x2​ con x1<x2x_1 < x_2x1​<x2​ allora anche f(x1)≤f(x2)f(x_1) \leq f(x_2)f(x1​)≤f(x2​)
  • Funzioni decrescenti se scegliendo x1x_1x1​ e x2x_2x2​ con x1<x2x_1 < x_2x1​<x2​ allora f(x1)≥f(x2)f(x_1) \geq f(x_2)f(x1​)≥f(x2​)

Si dice strettamente crescente o decrescente se è al posto di ≤\leq≤ e ≥\geq≥ c’è <<< e >>>

Proprietà funzioni monotone

  • Se funzione strettamente monotono allora è iniettiva.
  • Se funzione iniettiva può non essere monotona.
  • La funzione è costante se l’immagine ha un solo elemento.

Funzione inversa

La funzione inversa di f si indica con f−1f^{-1}f−1 ed è la legge che associa ad ogni punto y un unico punto x. Per cui una funzione si può invertire solo se è biunivoca.

Es. 1

f(x)=3xf(x) = 3xf(x)=3x

Per trovare l’inverso isoliamo la x:

y3=x\frac{y}{3} = x3y​=x

ora invertiamo la x con la y:

y=x3→f−1(x)=x3y = \frac{x}{3} \rightarrow f^{-1}(x) = \frac{x}{3}y=3x​→f−1(x)=3x​

Es. 2

f(x)=x3f(x) = \sqrt[3]{x}f(x)=3x​

y3=xy^3 = xy3=x

y=x3→f−1(x)=x3y = x^3 \rightarrow f^{-1}(x) = x^3y=x3→f−1(x)=x3

Proprietà funzioni inverse

  • L’immagine della funzione inversa è il dominio della funzione
  • (f−1∘f)(x)=f−1(f(x))=x(f^{-1} \circ f)(x) = f^{-1}(f(x)) = x(f−1∘f)(x)=f−1(f(x))=x
  • (f∘f−1)(y)=f(f−1(x))=y(f \circ f^{-1})(y) = f(f^{-1}(x)) = y(f∘f−1)(y)=f(f−1(x))=y
  • (f−1)−1=f(f^{-1})^{-1} = f(f−1)−1=f

Simmetrie

Una funzione è detta:

  • pari se i valori di x sono simmetrici rispetto all’asse y e quindi f(−x)=f(x)f(-x) = f(x)f(−x)=f(x)
  • dispari se i valori di x sono simmetrici rispetto all’origine degli assi f(−x)=−f(x)f(-x) = -f(x)f(−x)=−f(x)
  • Se non rispetta nessuno dei valori precedenti allora è né pari né dispari

Funzioni periodiche

Traslazioni, dilatazioni e riflessioni

Tramite delle operazioni è possibile modificare le funzioni:

  • Traslazione: spostare tutti i punti della funzione nelle quattro direzioni utilizzando un vettore.
  • Dilatazione:
    • orizzontale: i punti si allontanano all’asse delle y.
    • verticale: i punti si allontanano dall’asse delle x.
  • Contrazione
    • orizzontale: i punti si avvicinano all’asse delle y.
    • verticale: i punti si avvicinano dall’asse delle x.
  • Riflessione: specchiare la funzione secondo una determinata retta.
